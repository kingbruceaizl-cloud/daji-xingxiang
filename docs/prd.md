# 大吉形象 PRD

## 1. 产品概述

大吉形象是一款 AI 形象设计产品，帮助形象顾问、运营人员或商家基于客户上传的图片/视频，快速完成发型、妆容、服装、鞋包、饰品等搭配，并生成可交付的形象图片和变装短视频。

MVP 第一阶段不做 Lovart 式复杂画布，而是做“形象大师无画布版”：用步骤式流程降低使用门槛，让用户按顺序上传素材、选择风格和商品、生成结果。

## 2. 产品目标

- 让非技术人员可以完成客户形象设计。
- 将商品库和 AI 生图/视频能力打通。
- 支持真实商品和搭配素材两种商品类型。
- 后台可配置模型、商品、脚本、视频模板、音乐。
- 为未来 Lovart 风格画布版预留项目、素材、任务和模型抽象。

## 3. MVP 范围

### 本期要做

- 无需登录即可创建本地形象项目并进入形象大师流程。
- 保留登录、注册、退出模板，为后续 Supabase 上线和多员工权限做准备。
- 形象设计项目创建。
- 客户图片/视频上传。
- 商品库选择：发型、妆造、衣服、裤子、鞋子、手饰/饰品等。
- 风格模板选择。
- AI 形象图片生成任务。
- AI 变装短视频生成任务。
- 生成任务状态查看。
- 生成结果保存到素材库。
- 后台管理商品、分类、风格模板、视频脚本、音乐和模型配置。

### 本期不做

- Lovart 复杂画布。
- React Flow 节点编排。
- 支付、积分、会员、额度系统。
- 多员工、多角色权限。
- 强制登录和客户授权流程。
- 真人肖像授权确认流程。
- 完整长视频剪辑器。

## 4. 用户角色

### 项目负责人

第一阶段唯一后台使用者。可在本地或演示模式下管理模型、商品、模板、脚本、音乐和生成任务；上线后再接入登录和角色权限。

### 形象顾问

后续角色。可为客户创建形象项目、上传素材、选择商品、生成图片和视频。

### 客户

后续可选角色。第一阶段不直接登录系统，只作为被设计对象。

## 5. 核心流程

### 形象设计流程

1. 创建形象项目。
2. 上传客户正面照、半身照、全身照或视频。
3. 选择目标风格，例如通勤、约会、轻奢、国风、韩系、商务、婚礼、短视频种草等。
4. 选择商品和素材，包括发型、妆容、上衣、裤子、裙装、鞋子、包、饰品。
5. 系统根据客户素材、风格模板和商品信息拼接生成提示词。
6. 发起生图任务。
7. 选择满意图片后发起变装短视频任务。
8. 保存、下载或进入后续长视频组合。

### 后台配置流程

1. 配置 AI 模型通道和模型。
2. 创建商品分类。
3. 上传商品图、素材图或真实商品。
4. 配置风格模板和提示词片段。
5. 配置视频脚本模板。
6. 配置音乐。
7. 查看生成任务和失败原因。

## 6. 页面清单

- `/`: 首页或跳转页。
- `/protected`: 登录后首页，展示形象大师入口、最近项目。
- `/projects`: 项目列表。
- `/projects/new`: 创建形象项目。
- `/projects/[id]`: 项目详情与生成结果。
- `/studio/[projectId]`: 无画布形象大师工作流。
- `/admin`: 后台总览。
- `/admin/products`: 商品库管理。
- `/admin/styles`: 风格模板管理。
- `/admin/video-templates`: 视频模板管理。
- `/admin/music`: 音乐库管理。
- `/admin/models`: 模型配置。
- `/admin/jobs`: 生成任务记录。

## 7. Lovart 参考取舍

参考 Lovart 的：

- 左侧或侧边栏导航结构。
- 首页大输入框和最近项目入口。
- 灵感发现瀑布流。
- 项目/素材/生成任务的组织方式。

不参考 Lovart 的：

- 品牌名、视觉符号和文案。
- 完整画布作为 MVP 主流程。
- 复杂节点式创作体验。

## 8. AI 模型策略

第一阶段要使用统一模型通道抽象，不把业务写死到某一家模型。

候选模型通道：

- KIE
- OpenAI
- 即梦
- 可灵
- 通义

模型能力按任务类型归类：

- 文字生成：形象方案、提示词、脚本文案和商品描述。
- 图片理解：客户照片、商品图和参考图识别。
- 文生图
- 图生图
- 图生视频
- 视频生成
- 长视频合成
- 文案生成

后台需要提供“任务能力路由”配置，让不同操作默认调用不同模型：

- 生成形象方案和脚本：走文字生成模型。
- 分析客户图片和商品图片：走图片理解模型。
- 未上传客户素材时生成图片：走文生图模型。
- 基于客户照片生成形象图：走图生图模型。
- 基于形象图生成变装短视频：走图生视频模型。
- 长视频后续扩展：走长视频生成模型。

## 9. 商品库策略

商品分两种：

- 真实可售商品：有价格、库存、购买链接、商品编码。
- 搭配素材：只用于形象生成，没有真实售卖信息。

商品都应支持：

- 分类
- 标签
- 主图
- 参考图
- 提示词描述
- 适用风格
- 是否上架
