# 大吉形象本地一键启动

## 最简单的使用方式

在项目根目录双击：

```text
启动大吉形象.command
```

启动文件会自动完成下面的事情：

1. 进入当前项目目录。
2. 检查 Node.js、pnpm 和项目依赖。
3. 如果依赖不存在，自动安装依赖。
4. 自动构建当前最新页面。
5. 从 3000 端口开始寻找可用端口。
6. 在后台启动大吉形象本地服务。
7. 等待页面可访问后，自动用默认浏览器打开。

启动文件使用 macOS 自带的后台服务管理。终端窗口关闭后，本地服务仍会继续运行。

## 停止本地服务

在项目根目录双击：

```text
停止大吉形象.command
```

停止文件只会结束由本项目一键启动文件创建的服务，不会关闭其他项目。

## 第一次无法双击时

macOS 可能拦截第一次打开本地脚本。遇到提示时：

1. 在 Finder 中右键点击 `启动大吉形象.command`。
2. 选择“打开”。
3. 在确认窗口中再次选择“打开”。

以后可以直接双击。

## 运行记录

一键启动会在项目中生成下面的本地文件，它不会提交到 Git：

- `.daji-local.port`：当前使用的端口。

运行日志保存在 macOS 标准日志目录：

```text
~/Library/Logs/DajiXingxiang/
```

如果启动失败，启动窗口会直接显示最后一段日志，方便排查。
